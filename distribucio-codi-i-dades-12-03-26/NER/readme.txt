NER
Version v1.0
Date March 12, 2026

This folder contains the code and data to run an example of Named Entity Recognition (NER) using small LLMs.
To know how to use this content, follow the instructions in the project description document:
/doc/MGL-GIA-enunciat-practica-en-grup-ABSA.publicada-raco-12-03-26.pdf
You can adapt the scripts distributed here and apply them to Aspect Based Sentiment Analysis, which is the real problem you have to address in the course project. 

Folders within NER:
/bin : scripts and Python code to run the experiments on NER
/data: training and development sets
