ABSA
Version	v1.0
Date March 12, 2026

This folder contains the datasets for the group project on Aspect Based Sentiment Analusis (ABSA).
The bin/ subfolder is empty intentionally. You need to adapt the scripts and Python code from the NER problem (available at /NER/bin/*) to ABSA. To know how to do it, follow the instructions in the project description document:
/doc/MGL-GIA-enunciat-practica-en-grup-ABSA.publicada-raco-12-03-26.pdf

Folders	within ABSA:
/bin : empty
/data: training and development sets
